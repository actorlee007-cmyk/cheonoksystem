# [네이버 블로그용 / AI 자동화 운영일지] AI한테 "블로그 자동화 시스템"을 직접 만들어 달라고 해봤습니다

요즘 유튜브를 보면 "AI로 블로그 자동화해서 애드센스/애드포스트 수익화"
영상이 정말 많죠. 저도 궁금해서, 직접 AI(Claude Code)한테 "이런 자동화
구조를 코드로 만들어줘"라고 시켜봤습니다. 오늘은 그 과정을 그대로
기록해봅니다.

## 오늘 한 일을 한 줄로 요약하면

네이버 블로그/티스토리에 올릴 글의 주제·구성·CTA까지 자동으로 뽑아주는
파이썬 스크립트를 만들었습니다.

## 왜 필요했나

저는 원래 주식 모의투자(페이퍼트레이딩) 자동화 시스템을 만들고
있었는데, 그 작업을 하면서 "이 작업 일지 자체가 콘텐츠가 될 수
있겠다"는 생각이 들었어요. 그런데 매번 "오늘 뭘 썼더라... 제목은 어떻게
잡지... 마지막에 뭘 넣어야 하지"를 고민하는 게 은근히 시간이 들더라고요.
그래서 이 고민 자체를 자동화하기로 했습니다.

## 어떻게 풀었나 (쉽게 풀어서)

- 먼저 "어떤 주제로 쓸지"를 4가지 카테고리로 나눴어요: ① 오늘의 자동화
  작업일지, ② 자동화 툴 따라하기 가이드, ③ 모의투자 리포트 해설(투자
  권유 아님), ④ 사이드 프로젝트 솔직 후기.
- 각 카테고리마다 "제목 - 도입부 - 목차(소제목 5개) - 마무리 CTA" 틀을
  만들어뒀습니다.
- 네이버 검색 API가 연결되어 있으면, 같은 키워드로 지금 상위노출되는 글
  제목을 가져와서 "요즘 이런 제목이 잘 먹히는구나"를 참고하게 했고, API가
  없거나 막혀도 멈추지 않고 기본 틀로 글감을 만들어주게 했어요.
- 결과물은 JSON/CSV/마크다운 3가지로 나오는데, 그중 마크다운 파일을
  열면 바로 복사해서 블로그에 붙여넣을 수 있는 형태입니다. (지금 이
  글도 그렇게 만들어진 초안을 사람이 다듬은 버전입니다.)

## 결과 & 다음 할 일

지금은 "글의 뼈대"까지만 자동으로 나옵니다. 실제 본문은 사람이 직접
채워야 해요 — 오히려 이게 "진짜 사람이 쓴 글"이라는 신호이기도 하고요.
다음 할 일은 ① 네이버 오픈 API 키를 발급받아 연결하기, ② 이 글을
포함해서 실제로 며칠간 꾸준히 포스팅해보고 애드포스트 심사가 어떻게
되는지 확인하기입니다. 결과가 어떻든(되든 안 되든) 다음 글에 그대로
적을 생각이에요.

## 같은 고민하는 분께 팁 1개

"AI한테 글을 통째로 써달라"보다는, "이런 구조의 틀을 코드로 만들어줘"라고
요청하는 게 훨씬 결과물이 좋았습니다. 제목/구성/CTA 같은 '뼈대'는 AI가
일관되게 잘 뽑아주고, 그 안에 들어가는 '진짜 경험'은 결국 본인이 채워야
콘텐츠에 힘이 생기더라고요.
