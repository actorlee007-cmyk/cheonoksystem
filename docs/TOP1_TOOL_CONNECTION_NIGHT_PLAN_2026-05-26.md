# JOS Top-1 Tool Connection Night Plan — 2026-05-26

## Purpose
This is the prioritized tool connection plan for turning JOS/CHEONOK from a chat-centered system into an execution-grade operating system.

The goal is not to connect every app blindly. The goal is to connect tools that close the loop:
Observe -> Remember -> Decide -> Execute -> Collect Revenue -> Deliver -> Measure -> Upgrade.

## Core Rule
A tool is connected only if it strengthens at least one of these:
1. revenue loop
2. execution automation
3. customer/lead ledger
4. content production
5. data analysis
6. operational memory
7. trust/security/evidence
8. CEO reporting
9. exit-grade asset structure

## Final Veto
Do not connect tools that:
- require raw secrets in chat
- duplicate an existing core tool without clear improvement
- increase manual workload
- create legal/payment/privacy/account risks
- cannot be mapped to CASH CHAIN or the five business lines
