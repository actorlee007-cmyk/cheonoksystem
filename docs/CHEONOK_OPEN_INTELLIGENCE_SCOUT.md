# CHEONOK Open Intelligence Scout

상태: ACTIVE / 정본 필터 기반 / 무료 오픈지식·도구 흡수 엔진

## 0. 목적

CHEONOK Open Intelligence Scout는 전 세계에 공개된 오픈소스, 무료 도구, 논문, 데이터셋, 자동화 프레임워크, 콘텐츠 제작 도구, AI 모델, 사업/마케팅 지식을 정본 기준으로 탐색·분류·평가·흡수하기 위한 시스템이다.

핵심 목적은 단순 자료 수집이 아니다.

```text
전 세계 오픈 지식
↓
CHEONOK 정본 필터
↓
PASS / HOLD / BLOCK
↓
도구·데이터·콘텐츠·자동화·수익모델로 흡수
↓
시스템 자율성장
```

## 1. 최상위 원칙

정본 기준으로 모든 후보는 다음 질문을 통과해야 한다.

```text
1. 대표의 시간을 줄이는가?
2. 지금 현실 생존 또는 현금흐름에 도움이 되는가?
3. 콘텐츠·유입·보고·자동화·데이터·투자판단 중 어디에 붙는가?
4. 무료 또는 저비용인가?
5. 설치 난이도와 유지보수 난이도가 감당 가능한가?
6. 보안 위험이 낮은가?
7. CHEONOK의 동반성장·사람가치 증폭 철학과 충돌하지 않는가?
8. 64GB 업그레이드 이후 활용 가치가 커지는가?
9. 실패해도 학습/패치 자산으로 남는가?
10. 후퇴 없이 시스템에 누적 가능한가?
```

## 2. PASS / HOLD / BLOCK 판정

### PASS

즉시 흡수 또는 실험 가능.

조건:

```text
- 현금흐름, 유입, 콘텐츠, 보고, 자동화 중 하나에 직접 연결
- 무료 또는 저비용
- Windows 운영 가능
- 보안 위험 낮음
- 설치/실험 실패 시에도 학습 자산으로 남음
```

### HOLD

좋지만 지금은 보류.

조건:

```text
- GPU/RAM/설치 난이도 부족
- 현재 생존 현금흐름과 직접 연결 약함
- 좋은 아이디어지만 당장 운영에 부담
- 64GB 이후 재검토 필요
```

### BLOCK

폐기 또는 접근 금지.

조건:

```text
- 보안 위험 높음
- 비밀번호/API 키 탈취 가능성
- 불법/저작권 위험
- 과도한 리소스 사용
- 대표의 시간을 오히려 잡아먹음
- 정본 철학과 충돌
```

## 6. 현재 1차 우선순위

현재 CHEONOK의 병목은 다음이다.

```text
유입 부족
콘텐츠 부족
고객 데이터 부족
반복수익 부족
로컬 AI 안정성 부족
```

따라서 1차 스카우트 우선순위는 다음 순서다.

```text
1. 콘텐츠 제작/영상 편집 도구
2. 자동 자막/스크립트 도구
3. 무료 진단/고객 수집 폼 도구
4. 보고서 자동화 도구
5. 로컬 에이전트/자동화 런타임
6. AI 모델/데이터셋
7. 대형 리서치/논문 엔진
```

```text
보안 위험: 저장소 확인 전 HOLD
판정: PASS 후보 / 저장소 확인 후 최종 판정
다음 행동: GitHub URL 확보 후 README, 설치법, 라이선스, Windows 지원 여부 확인
```

## 8. 운영 원칙

```text
도구를 많이 모으는 것이 목적이 아니다.
대표의 시간을 줄이고, 현금흐름과 사람가치 증폭에 연결되는 것만 흡수한다.
무료라도 복잡하면 HOLD.
멋져도 지금 돈과 성장에 연결되지 않으면 HOLD.
위험하면 BLOCK.
작고 실전적인 것부터 PASS.
```

## 9. 최종 정의

CHEONOK Open Intelligence Scout는 전 세계 오픈 지식과 무료 도구를 무작정 수집하는 시스템이 아니라, 정본 필터를 통과한 것만 현실 생존·현금흐름·콘텐츠 유입·자동화·구독·사람 가치 증폭에 연결하는 성장 흡수 엔진이다.

현재 상태: ACTIVE / 1차 대상 OpenCut 및 콘텐츠 유입 도구군 / 64GB 업그레이드 이후 실행 엔진 확대.
