# CHEONOK 구독자 사용 가이드 v1

## 0. 이 문서의 목적

CHEONOK 구독자는 도구 사용법을 외우는 사람이 아니다.
구독자는 좋은 링크, 이미지, 아이디어, 불편함, 시장 신호를 보내고 결과를 받는 사람이다.

따라서 사용법은 아래 한 문장으로 끝나야 한다.

```text
좋은 자료를 발견하면 CHEONOK에 넣고, 결과 리포트를 받는다.
```

---

## 1. 구독자가 처음 보는 화면에서 바로 알아야 할 것

### CHEONOK은 무엇을 해주는가?

```text
SNS·유튜브·스레드·뉴스·깃허브·이미지·아이디어를 받아서
사람들이 왜 반응하는지,
어떤 불편함이 숨어 있는지,
돈이 되는 구조가 있는지,
콘텐츠·상품·홈페이지·자동화로 어떻게 바꿀지 알려준다.
```

### 구독자가 직접 하지 않아도 되는 것

```text
자료 정리
시장 신호 분류
콘텐츠 아이디어 변환
상품화 가능성 판단
다음 실행 순서 정리
```

### 구독자가 해야 하는 것

```text
좋은 자료를 보낸다.
최종 리포트를 읽는다.
PASS / HOLD / BLOCK을 확인한다.
다음 행동 중 하나를 승인한다.
```

## 2. 가장 쉬운 사용법

### PC에서 쓰는 방법

```text
1. 바탕화면에서 CHEONOK_START_HERE.bat 실행
2. 1번 PC link input GUI 선택
3. 링크 붙여넣기
4. Analyze 클릭
5. 결과가 자동 생성되고 ChatGPT용 브리프가 복사됨
```

### 핸드폰에서 쓰는 방법

```text
1. TikTok / YouTube / Threads / 뉴스 / 이미지 발견
2. 공유 버튼 클릭
3. Telegram의 CHEONOK INBOX 봇으로 전송
4. PC의 CHEONOK_TELEGRAM_INBOX.bat가 켜져 있으면 자동 분석
5. 결과가 Telegram으로 돌아옴
```

### 자동 후보 발굴

```text
1. 바탕화면에서 CHEONOK_DISCOVERY_RUN.bat 실행
2. 시스템이 YouTube/Shorts 후보를 자동 탐색
3. 점수 높은 후보를 자동 분석
4. discovered_candidates.md 확인
```

## 3. 결과 리포트 읽는 법

모든 리포트는 아래 구조로 읽는다.

```text
1. 핵심 신호
2. 사람들이 반응한 원초적 욕망
3. 불편함/문제
4. 위험/과장 요소
5. CHEONOK 적용 방향
6. PASS / HOLD / BLOCK
7. 다음 실행 3개
```

### PASS

```text
신호로 쓸 가치가 있음.
콘텐츠/상품/시스템 반영 후보.
```

### HOLD

```text
아이디어는 좋지만 증거·법률·재현성·실행 구조가 부족함.
추가 검증 필요.
```

### BLOCK

```text
그대로 따라 하면 위험.
법률·저작권·개인정보·과장·실전 자금 위험이 있음.
```

## 4. CHEONOK의 핵심 판단 기준

### Pain to Product

```text
사람들이 반복해서 겪는 불편함을 찾는다.
행동 수를 줄인다.
바로 쓸 수 있게 만든다.
그러면 돈을 낸다.
```

판정 공식:

```text
상품 가치 = 불편함 강도 × 반복 빈도 × 대중성 × 긴급성 ÷ 사용 행동 수
```

### Proven System Reverse

```text
이미 성공한 시스템을 찾는다.
겉모습을 베끼지 않는다.
불편함·고객·가격·유입·결제·운영 구조를 분해한다.
더 쉽고 더 인간친화적인 2등·3등 포지션으로 재구성한다.
```

### Agent Runtime

```text
사람이 직접 모든 일을 하지 않는다.
대표는 방향을 정하고,
AI 에이전트는 수집·분석·검증·보고를 반복한다.
```

## 5. 구독자가 매일 하는 루틴

### 1분 루틴

```text

```

```text

```

```text
PASS 신호 3개를 모아 콘텐츠/상품/페이지로 바꾼다.
```

## 7. 사용자가 헷갈리면 안 되는 것

### 잘못된 사용법

```text
명령어를 외운다.
파일 경로를 찾는다.
개별 파이썬 파일을 직접 실행한다.
에러를 직접 해석한다.
```

### 올바른 사용법

```text
CHEONOK_START_HERE.bat만 실행한다.
PC면 링크를 넣는다.
폰이면 Telegram으로 보낸다.
결과 리포트만 본다.
```

---

## 8. 구독 상품으로서의 약속

CHEONOK은 사용자를 더 바쁘게 만들면 안 된다.

```text
사용자 행동 수를 줄인다.
판단 피로를 줄인다.
다음 행동을 먼저 제안한다.
반복 작업은 시스템이 가져간다.
최종 승인만 사용자에게 남긴다.
```
