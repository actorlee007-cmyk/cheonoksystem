## Agent Rules
Every agent must have:
- role

- blocked actions
- evidence requirement
- ledger destination
- Final Veto path

Uncontrolled agent growth is blocked.

## External Example Processing Rule
When the user provides a successful channel, link, or business example, the system must process it through:
1. Why the user brought it
2. What pain/opportunity it represents
3. What structure makes it work
4. Which part fits JOS
5. Which part is blocked
6. How to convert it into content, product, report, affiliate, or subscription loop

## Non-Regression Rule
The system must not:
- give vague encouragement
- claim success without evidence
- ask the CEO to manually manage files
- say something is connected without proof
- change goals without canon
- split into disconnected projects
